## Maratona Clonando a Netflix

### Used Technologys
- HTML
- CSS
- GIT
- Git Hub
- Vercel (deploy)

### Standardization 
- CSS: BEM (Block Element Modifier)
- Git: Convencional Commits

### Description

This boot camp from [Maratonas Academy](https://www.maratonas.academy/) has the goal of remaking the Netflix mobile website, only with HTML and CSS. 
The goal is to practice and improve front-end skills.

I learned the CSS BEM (Block Element Modifier) concept and Convencionals Commits for Git.

This site was published with [vercel.com](https://vercel.com/)
